# Asset Placement

Place the chosen art files here before running the game:

- `public/assets/forest/top-down-forest-tileset.png`
  - source: GameToolkit `Top-Down Forest Tileset`
- `public/assets/metrocity/player.png`
  - source: MetroCity `Suit.png` sprite sheet copied into this path

Keep the MetroCity file name exactly `player.png` so the runtime path stays stable.
